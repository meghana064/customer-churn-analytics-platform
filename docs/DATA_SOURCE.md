# Data Source — IBM Telco Customer Churn Dataset

## Dataset Name
**IBM Telco Customer Churn Dataset**

## Description
This dataset contains customer demographic, account, service subscription, and billing information from a telecom provider, along with a churn label indicating whether the customer left the company.

## Business Problem
Telecom businesses want to **reduce customer churn** by identifying customers who are likely to cancel service. Predicting churn enables targeted retention actions (support interventions, contract upgrades, discounts, and proactive outreach).

## Number of Records
**7,043** customers (rows).

## Number of Features
**21** columns total in the common CSV version (including the identifier and the target).

## Target Variable
- **Churn**: whether the customer churned (`Yes/No`)

## Important Input Features
- Gender  
- SeniorCitizen  
- Partner  
- Dependents  
- Tenure  
- PhoneService  
- InternetService  
- Contract  
- MonthlyCharges  
- TotalCharges  
- PaymentMethod
